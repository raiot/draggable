# Draggable

Draggable is simple helper to move any HTML element to a position and drop it.

## Installation

```
npm install
```
## To see the demo

```
npm start
```

## Usage

```
import { bindDraggable } from 'dist/index.js';
const element = document.getElementById('the-id');
bindDraggable(element);
```
